Vilkår for bruk

Sist oppdatert: 2026-03-19

1. Om tjenesten

ActivityFinder er en nettside for å utforske og publisere aktiviteter. Noe innhold kan være brukergenerert, og kan endres eller fjernes.

2. Konto og tilgang

For enkelte funksjoner (f.eks. admin/administrasjon) kreves innlogging. Du er ansvarlig for aktiviteter som skjer via kontoen din.

3. Brukeratferd, analyse og logging

For å forbedre tjenesten og forstå hvordan siden brukes, samler vi inn teknisk bruksdata (analyse), som kan inkludere:

  • besøkte sider og handlinger (f.eks. klikk, navigasjon)
  • omtrentlig enhets-/nettleserinformasjon
  • tidspunkt og varighet

Dette spores via Vercel sin analyse-/loggingfunksjonalitet. Dataene brukes til innsikt og drift, ikke for å selge persondata.

4. Informasjonskapsler (cookies) og lokal lagring

Nettsiden kan bruke lokal lagring (localStorage) og/eller informasjonskapsler for å holde deg innlogget og for grunnleggende funksjonalitet. Analyseverktøy kan også bruke tilsvarende teknologier.

5. Tredjeparter

Tjenesten kan bruke tredjepartsleverandører for drift (f.eks. hosting/analyse via Vercel og databaser/autentisering via Supabase). Disse leverandørene kan behandle tekniske data for å levere tjenesten.

6. Ansvarsbegrensning

Tjenesten leveres “som den er”. Vi prøver å holde informasjon oppdatert, men kan ikke garantere at innhold alltid er korrekt. Bruk skjer på eget ansvar.

7. Endringer

Vi kan oppdatere vilkårene ved behov. Datoen øverst viser når vilkårene sist ble endret.

8. Kontakt

Har du spørsmål om vilkår eller sporing/analyse, ta kontakt med oss via kanalene oppgitt på nettsiden.
